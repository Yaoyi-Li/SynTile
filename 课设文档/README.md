# SynTile
implementation and improvement of Synthesis of Tiled 
Patterns Using Factor Graphs

在Matlab中将工作目录设置为当前目录，输入synTiles运行，
即可执行该算法，程序将使用默认参数和样本数据进行计算，
若需修改参数及样本数据参见 课设文档>课设程序使用手册 
2.1及2.2节具体说明。

Matlab在执行完数据预处理后将显示preprocessing finished，
对于每次迭代将显示迭代序号，并对局部最优解和全局最优
解进行提示。执行过程中在设计样本目录下会输出每次迭代
产生的设计方案，最优结果bestDesign.jpg将输出在当前目
录。
